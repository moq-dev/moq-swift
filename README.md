# Moq (Swift Package)

Auto-generated mirror of the Swift package for [Media over QUIC](https://github.com/moq-dev/moq).

Source, issues, and pull requests live in [moq-dev/moq](https://github.com/moq-dev/moq); this repo only carries tagged Swift Package Manager releases.

## Install

```swift
.package(url: "https://github.com/moq-dev/moq-swift", from: "0.2.0"),
```

Each tagged release here depends on a prebuilt `MoqFFI.xcframework` attached to the matching `moq-ffi-v*` release on the source repo.

See [moq-dev/moq/swift/README.md](https://github.com/moq-dev/moq/blob/main/swift/README.md) for usage, local development, and the release process.

This repo's contents are regenerated by [`swift/scripts/publish.sh`](https://github.com/moq-dev/moq/blob/main/swift/scripts/publish.sh) on every `moq-ffi-v*` tag, so any direct commits will be overwritten.
